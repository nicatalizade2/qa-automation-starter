# QA Automation Starter (Python + Playwright)

This repository contains my first UI automation framework built with:

- Python
- PyTest
- Playwright (pytest-playwright)
- Page assertions using Playwright `expect`

### DemoQA UI Tests
- Homepage title validation
- Elements menu visibility
- TextBox form submission verification

## How to Run

1. Install dependencies:
pip install -r requirements.txt
playwright install
